getCachedApiLogo:size:aspectRatio:bg_color:operatorName:language:

This function return the image url requested that fits the params provided. Null params will be ignored in the search

- (UIImage *)getCachedApiLogo:(NSString *)api size:(logoSizeTypeEnum)size aspectRatio:(aspect_ratioTypeEnum)aspectRatio bg_color:(bg_colorTypeEnum)bg_color operatorName:(NSString *)operatorName language:(NSString *)language

Parameters

api

Conditional parameter to check in the logo stored item to return an specific logo url image. Types availables: ‘PAYMENT’,‘OPERATORID’

size

Conditional parameter to check in the logo stored item to return an specific logo url image. Types availables: ‘SMALL’,‘MEDIUM’,‘LARGE’

aspectRatio

Conditional parameter to check in the logo stored item to return an specific logo url image. Types availables: ‘SQUARE’,‘LANDSCAPE’.

bg_color

Conditional parameter to check in the logo stored item to return an specific logo url image.Types availables: ‘NORMAL’,‘REVERSED’,‘BLACK’

operatorName

Operator Conditional parameter to check in the logo stored item to return an specific logo url image

language

Conditional parameter for set the language if available

Object representation of the result stored in the last success getLogo request made.
