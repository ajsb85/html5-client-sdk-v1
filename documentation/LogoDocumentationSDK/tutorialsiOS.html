iOS Tutorial

Here is an example illustrating the simplest way to use LogoSDK on iOS.

Including Logo library

To start we need to import the Logo library in to the project (refer to Getting Started). 

#import "LogoAPI.h"

Creating Logo object

To invoke the Discovery functions its needed to set up first a Discovery object. We need also to assign a delegate class which will be the receiver class for the calls responses (in the example below will use self) 

//API object creation
LogoAPI *logoObj = [[LogoAPI alloc] init];

//Delegate assignments
logoObj.logoDelegate = self;
logoObj.traceMode=TRUE;

Executing Logo functions

The Logo library implement the following functions.

  • Get logo automatic mnc mcc

    This functions make a request to obtain a Logo information. In a wrong case the error will be returned. 

    //Get Logo call
[logoObj getLogo:@"https://integration-dt.apiexchange.org/v1/logo" logoSize:logoSizeTypeEnumMEDIUM aspectRatio:aspect_ratioTypeEnumLANDSCAPE bg_color:bg_colorTypeEnumBLACK ipAddress:nil];

  • Get logo manual mnc mcc

    This functions make a request to obtain a Logo information. In a wrong case the error will be returned. 

    //Get Logo call
[logoObj getLogo:@"https://integration-dt.apiexchange.org/v1/logo" logoSize:logoSizeTypeEnumMEDIUM aspectRatio:aspect_ratioTypeEnumLANDSCAPE bg_color:bg_colorTypeEnumBLACK ipAddress:nil mcc:@"1234" mnc:@"4321"];

  • Get cached Logo item

    This functions get the last success Logo request cached information. 

    //Get cached Logo call
[logoObj getCachedLogoItem]

  • Get cached API Logo item

    This functions look for in the last success Logo request cached information. The result will be an UIImage object. 

    //Get cached API Logo call
UIImage *imageLogo = [logoProvider getCachedApiLogo:@"operatorid" size:logoSizeTypeEnumSMALL aspectRatio:aspect_ratioTypeEnumLANDSCAPE bg_color:bg_colorTypeEnumNORMAL operatorName:@"exchange" language:@"en"];

  • Clear cached Logo item

    Clear the last stored Logo item. 

    //Clear Cached Logo
NSLog(@"Cleared cache logo item?: %@",([logoObj clearCachedLogoItem] ? @"YES" : @"NO"));

Managing Logo response

The Logo function use asynchronous request internally. Due to this a formal protocol is defined to manage the Logo functions response. 

#import "LogoAPI.h.h"
@interface GenericClass : NSObject <ProcessGetLogoResponseDelegate>

The protocol implementation force us to declare the getLogoSuccessful, getLogoError functions which will receive the get Discovery responses. 

@implementation GenericClass
// FORMAL PROTOCOL IMPLEMENTATION
-(void)getLogoSuccessful:(long)statusCode data:(NSArray *)data{
    NSLog(@"logo successful");
    NSLog(@"statusCode - %ld",statusCode);
    NSLog(@"result - %@",data);
}
-(void)getLogoError:(long)statusCode data:(NSMutableDictionary *)errorData{
    NSLog(@"logo error");
    NSLog(@"statusCode - %ld",statusCode);
    NSLog(@"result - %@",errorData);
}
