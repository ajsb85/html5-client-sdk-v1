Installing LogoSDK on iOS

To use LogoSDK, just add the archive files called libGSMA IOS SDK-Universal_i386-x86_64.a to your Xcode project within Frameworks folder. You are ready to start but remember that you need to include in your project the header files: "LogoAPI.h" and "Utils.h".

Example:
To use the logo functions you need to import the headers files to your code

1
 
#import  "LogoAPI.h"
