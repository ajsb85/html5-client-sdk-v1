Installing DiscoverySDK on Android

To use the library you need to add two jar files. For one, the archive files called com.gsma.android.oneapi.discovery.jar and on the other, the Jackson JSON library that is required for DiscoverySDK work well (DiscoverySDK tests are conducted with jackson-all-1.9.11.jar version). Just add the two jar files to your android project within libs folder. You are ready to start but remember that you need to import in your java files the classes that you use in it.

Example:
If you need to use DiscoveryProvider object in your class, you need to import it.

1
 
import com.gsma.android.oneapi.discovery.DiscoveryProvider;
